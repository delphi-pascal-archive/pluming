ВНИМАНИЕ!!!!!!!!
****************
Перед запуском активируйте REG-файл,
иначе программа закроется без слов.

****************
Программа поддерживает возможность добавления уровней.
Уровни хранятся в реестре.

Ключ: HKEY_LOCAL_MACHINE\Software\Vodoprovod
Параметры: Levels - отвечает за количество уровней в игре.
Субключи : LevelX, где X - номер уровня
В них параметры:
StartX, StartY - позиция X и Y старта, может принимать значения 0..9
FinishX, FinishY - то же, только финиша.
Time - время

****************

ВНИМАНИЕ!!!!!!!!!!!
Если параметр levels будет меньше количества уровней, то остальные уровни не будут обработаны,
а если больше, то при недостающем уровне программа закроется.
